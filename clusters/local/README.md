# clusters/local

Flux-managed manifests for the local k0s cluster.

## Layout

```
clusters/local/
  flux-system/       ← generated by `just flux-bootstrap`, do not edit manually
  infrastructure/    ← infra-level kustomizations (ingress, cert-manager, …)
  apps/              ← application kustomizations
```

## Bootstrap

```sh
export GITHUB_TOKEN=<your PAT with repo read/write>
just up            # start the k0s cluster
just flux-bootstrap
```

Re-running `just flux-bootstrap` is safe — Flux bootstrap is idempotent.
